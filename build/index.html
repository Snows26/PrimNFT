O QUE É ESCOLA NFT?

O grande problema de jogos como Axie Infinity, é que o jogo não disponibiliza personagens de graça para você começar no jogo, você é obrigado a fazer um investimento inicial pra começar a jogar e esse investimento costuma ser muito alto, os valores para a montagem de um time de Axies (composto por 3 personagens) chegam em cerca de US$ 271, grande parte dos interessados não consegue esse investimento para jogar. Com esse problema surgiram as escolas NFT’s, antes de tudo, escolas NFT’s não são escolas onde há um professor ensinando aluno, é apenas um termo utilizado para um modelo de negócio, que consiste em uma “ponte” que é feita entre duas pessoas, uma delas tem o capital para a compra dos personagens mas não possui o tempo para estar jogando com eles, já o outro lado da

“ponte” é representado por uma pessoa que não tem o capital inicial mas possui o interesse e tempo pra estar jogando, com os dois lados da ponte estabelecido, o investidor (pessoa que possui o investimento inicial) em cima de um contrato ele empresta a conta para o jogador (pessoa que não possui o investimento inicial) estar jogando para ele, conforme o jogador vai ganhando batalhas na conta do investidor, ele vai ganhando moedas do jogo, com essas moedas o investidor pode transformar em dinheiro mas por conta do contrato que foi estabelecido o investidor deve pagar o que foi definido em contrato com o jogador, os contratos podem ser de 50% para cada, 60% para o investidor e 40% para o jogador, tudo depende do que forcombinado antes.
